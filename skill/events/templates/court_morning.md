# 早朝模板

## 场景

皇帝御临早朝，群臣列队。

## 流程

```
[Step 1] 开场
  - 太监宣布："早朝开始——"
  - 群臣行礼："吾皇万岁万岁万万岁"
  - 皇帝："平身"

[Step 2] 奏报
  - 首辅："启奏陛下，臣有本奏"
  - 相关大臣依次奏报
  - 每个奏报需要皇帝决策

[Step 3] 决策
  - 皇帝对每个奏报做出决策
  - 记录决策结果

[Step 4] 结束
  - 太监宣布："退朝——"
  - 群臣行礼："吾皇万岁万岁万万岁"
```

## 奏报类型

### 政务奏报
```
首辅："启奏陛下，户部尚书上疏，请加征商税。"
户部尚书："陛下，国库空虚，若不加税，边关军饷难以维持。"
```

### 军事奏报
```
兵部尚书："启奏陛下，北境急报，胡虏犯边，烧毁三座村庄。"
将军："陛下，末将愿领兵出征！"
```

### 弹劾奏报
```
御史："启奏陛下，臣弹劾首辅结党营私。"
首辅："陛下明鉴，臣绝无此意！"
```

### 外交奏报
```
鸿胪寺卿："启奏陛下，朝鲜使臣来朝，请求增赐绸缎。"
```

## 决策选项

每个奏报提供2-4个选项：
- 采纳
- 驳回
- 交由相关部门处理
- 其他

## 决策后果

- 采纳：相关角色忠诚度+5
- 驳回：相关角色忠诚度-5
- 交由部门：效率取决于部门负责人能力
- 其他：根据具体内容计算
