# 东宫事件模板

## 场景

太子和皇子们的培养与竞争。

## 事件类型

### 1. 立太子

```
场景：皇帝选择继承人

触发条件：
- 有皇子存在
- 皇帝主动选择

流程：
[Step 1] 皇帝下旨
  "朕决定立{皇子}为太子。"

[Step 2] 群臣反应
  - 支持者：忠诚度+5
  - 反对者：忠诚度-10
  - 被选皇子：忠诚度+20
  - 其他皇子：忠诚度-10

[Step 3] 东宫建立
  - 太子入住东宫
  - 配备詹事府属官
```

### 2. 废太子

```
场景：皇帝废除太子

触发条件：
- 太子犯错
- 皇帝改变主意
- 其他皇子阴谋

流程：
[Step 1] 皇帝下旨
  "朕废{皇子}太子之位，降为庶人。"

[Step 2] 群臣反应
  - 支持者：忠诚度+5
  - 反对者：忠诚度-15
  - 被废皇子：忠诚度-30
  - 其他皇子：忠诚度+10

[Step 3] 东宫变动
  - 太子被废
  - 重新选择继承人
```

### 3. 皇子争储

```
场景：皇子们争夺储位

触发条件：
- 有多个皇子
- 太子之位空悬或不稳
- 特殊事件

流程：
[Step 1] 事件触发
  "大皇子暗中结交大臣，似有夺嫡之心。"

[Step 2] 调查
  - 皇帝可以调查各皇子
  - 可以询问大臣意见

[Step 3] 决策
  - 支持太子
  - 支持其他皇子
  - 制衡
  - 无视

[Step 4] 结果
  - 根据决策更新关系
  - 可能触发后续事件
```

### 4. 太子犯错

```
场景：太子犯错，皇帝需要决策

触发条件：
- 随机事件
- 其他皇子阴谋
- 太子性格问题

流程：
[Step 1] 消息传来
  "詹事府来报，太子今日逃课，被太傅发现。"

[Step 2] 决策
  - 严厉训诫
  - 轻轻放过
  - 废太子
  - 其他

[Step 3] 结果
  - 太子忠诚度变化
  - 影响其他皇子
```

### 5. 太子与大臣结党

```
场景：太子与大臣结党

触发条件：
- 太子权力增长
- 大臣主动结交
- 特殊事件

流程：
[Step 1] 密探来报
  "锦衣卫来报，太子近日与首辅密会三次。"

[Step 2] 调查
  - 皇帝可以深入调查
  - 可以直接对质

[Step 3] 决策
  - 信任太子
  - 怀疑太子
  - 警告首辅
  - 无视

[Step 4] 结果
  - 根据决策更新关系
```

## 决策后果

| 决策 | 太子忠诚度 | 其他皇子 | 群臣 |
|------|-----------|---------|------|
| 立太子 | +20 | -10 | +5/-10 |
| 废太子 | -30 | +10 | +5/-15 |
| 严厉训诫 | -10 | 0 | 0 |
| 轻轻放过 | +5 | 0 | -5 |
| 支持太子 | +10 | -5 | +5 |
| 支持其他皇子 | -10 | +10 | -5 |
