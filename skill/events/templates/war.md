# 战争事件模板

## 场景

与邻国的军事冲突。

## 战争流程

### 1. 战争触发

```
触发条件：
- 邻国关系 hostile
- 皇帝主动宣战
- 边境冲突升级

流程：
[Step 1] 兵部来报
  "兵部急报，北元大军犯边，已破三城！"

[Step 2] 皇帝决策
  - 应战
  - 求和
  - 逃跑（咸鱼帝专属）
```

### 2. 战前准备

```
流程：
[Step 1] 选择将领
  "陛下，末将愿领兵出征！"
  - 从将领中选择

[Step 2] 制定策略
  - 进攻
  - 防御
  - 奇袭
  - 僵持

[Step 3] 调配兵力
  - 选择出征军队数量
  - 留守兵力
```

### 3. 战斗过程

```
战斗计算：
  进攻方战力 = 军队数量 × 将领军事能力 × 策略加成
  防御方战力 = 军队数量 × 将领军事能力 × 策略加成

  胜负 = 进攻方战力 / 防御方战力

  结果：
  - 大胜（>1.5）：敌军溃败，占领城池
  - 小胜（1.0-1.5）：敌军撤退
  - 平局（0.8-1.0）：僵持
  - 小败（0.5-0.8）：我军撤退
  - 大败（<0.5）：我军溃败，失守城池
```

### 4. 战后处理

```
胜利：
[Step 1] 处理俘虏
  - 释放
  - 关押
  - 处决

[Step 2] 处理领土
  - 占领
  - 索要赔款
  - 要求和亲

[Step 3] 封赏将领
  - 升迁
  - 赏赐
  - 无表示

失败：
[Step 1] 处理败将
  - 问责
  - 宽恕
  - 处决

[Step 2] 处理敌国
  - 求和
  - 割地
  - 赔款

[Step 3] 稳定国内
  - 大赦
  - 减税
  - 无视
```

## 战争选项

### 进攻策略
```
优点：可能快速结束战斗
缺点：风险高，损失大
适合：军队数量优势时
```

### 防御策略
```
优点：损失小，稳固
缺点：被动，可能被围困
适合：军队数量劣势时
```

### 奇袭策略
```
优点：可能出其不意
缺点：风险极高，成功率低
适合：将领军事能力高时
```

### 僵持策略
```
优点：稳定，消耗敌方
缺点：时间长，经济压力大
适合：双方实力相当时
```
