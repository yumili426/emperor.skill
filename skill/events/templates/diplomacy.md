# 外交事件模板

## 场景

与邻国的外交互动。

## 事件类型

### 1. 使臣来朝

```
场景：邻国派遣使臣

流程：
[Step 1] 鸿胪寺来报
  "鸿胪寺来报，朝鲜使臣已到京城。"

[Step 2] 接见
  - 皇帝接见使臣
  - 使臣呈上国书和贡品

[Step 3] 决策
  - 接受朝贡，回赐
  - 拒绝朝贡
  - 提出要求
  - 无视

[Step 4] 结果
  - 关系变化
  - 可能触发后续事件
```

### 2. 朝贡请求

```
场景：邻国请求朝贡

流程：
[Step 1] 鸿胪寺来报
  "启奏陛下，安南国王遣使，请封为王。"

[Step 2] 决策
  - 同意册封
  - 拒绝册封
  - 提出条件
  - 无视

[Step 3] 结果
  - 关系变化
  - 可能影响其他邻国
```

### 3. 和亲请求

```
场景：邻国请求和亲

流程：
[Step 1] 鸿胪寺来报
  "启奏陛下，北元可汗遣使，请和亲。"

[Step 2] 决策
  - 同意和亲（送公主）
  - 拒绝和亲
  - 提出替代方案
  - 无视

[Step 3] 结果
  - 关系变化
  - 可能影响后宫
```

### 4. 战争威胁

```
场景：邻国发出战争威胁

流程：
[Step 1] 兵部来报
  "兵部急报，北元大军集结边境，似有进犯之意。"

[Step 2] 决策
  - 备战
  - 求和
  - 派使臣谈判
  - 无视

[Step 3] 结果
  - 关系变化
  - 可能触发战争
```

### 5. 战争请求

```
场景：邻国请求联合战争

流程：
[Step 1] 鸿胪寺来报
  "启奏陛下，朝鲜国王遣使，请联合抗倭。"

[Step 2] 决策
  - 同意联合
  - 拒绝联合
  - 提出条件
  - 无视

[Step 3] 结果
  - 关系变化
  - 可能触发战争
```

## 决策后果

| 决策 | 关系变化 | 其他影响 |
|------|---------|---------|
| 接受朝贡 | +10 | 经济+5 |
| 拒绝朝贡 | -10 | 经济-5 |
| 同意册封 | +15 | 文化+5 |
| 同意和亲 | +20 | 后宫变化 |
| 拒绝和亲 | -15 | 可能战争 |
| 备战 | 0 | 军事-10 |
| 求和 | +10 | 经济-10 |
| 同意联合 | +10 | 可能战争 |
