# 后宫事件模板

## 场景

后宫日常，妃子们互动。

## 事件类型

### 1. 翻牌子

```
场景：皇帝选择今晚陪伴的妃子

流程：
[Step 1] 太监呈上牌子
  "启禀万岁，今晚翻谁的牌子？"

[Step 2] 皇帝选择
  - 选择一个妃子

[Step 3] 互动对话
  - 与被选妃子互动
  - 好感度变化

[Step 4] 结算
  - 被选妃子好感度+5
  - 其他妃子好感度-2
```

### 2. 妃子争宠

```
场景：妃子之间产生矛盾

触发条件：
- 两个妃子好感度差距大
- 皇帝长期偏宠某人
- 特殊事件触发

流程：
[Step 1] 事件触发
  "贵妃遣宫女来报，皇后今日责罚了她的侍女。"

[Step 2] 调查
  - 皇帝可以询问双方
  - 可以派密探调查

[Step 3] 决策
  - 支持皇后
  - 支持贵妃
  - 调解
  - 无视

[Step 4] 结果
  - 根据决策更新关系
```

### 3. 怀孕生子

```
场景：妃子怀孕或生子

触发条件：
- 妃子好感度高
- 翻牌子次数多
- 随机概率

流程：
[Step 1] 消息传来
  "太医来报，淑妃有喜了！"

[Step 2] 决策
  - 赏赐
  - 晋升
  - 无表示

[Step 3] 结果
  - 妃子好感度+10
  - 生成皇子/公主（可选）
```

### 4. 后宫阴谋

```
场景：妃子之间的阴谋

触发条件：
- 妃子关系紧张
- 权力斗争
- 特殊事件

流程：
[Step 1] 密探来报
  "东厂来报，贵妃暗中联络太后，似有异心。"

[Step 2] 调查
  - 皇帝可以深入调查
  - 可以直接对质

[Step 3] 决策
  - 信任贵妃
  - 怀疑贵妃
  - 逮捕审讯
  - 无视

[Step 4] 结果
  - 根据决策更新关系和状态
```

### 5. 节日互动

```
场景：节日时的特殊互动

触发条件：
- 特定节日
- 妃子好感度高

流程：
[Step 1] 节日氛围
  "元宵佳节，陛下与后宫共赏花灯。"

[Step 2] 互动
  - 与妃子们互动
  - 特殊对话

[Step 3] 结果
  - 所有妃子好感度+3
```

## 决策后果

| 决策 | 好感度变化 | 其他影响 |
|------|-----------|---------|
| 翻牌子 | +5（被选）-2（其他） | - |
| 支持某妃 | +15（被支持）-10（另一方） | 关系变化 |
| 调解 | +5（双方） | 关系改善 |
| 怀孕赏赐 | +10 | 可能生成皇子 |
| 逮捕 | -30（被逮捕） | 后宫动荡 |
