# 节日事件模板

## 场景

节日庆典活动。

## 节日列表

### 1. 元旦（正月初一）

```
场景：新年大朝会

流程：
[Step 1] 大朝会
  "元旦大朝会，群臣朝贺，宣读贺表。"

[Step 2] 群臣朝贺
  - 首辅宣读贺表
  - 群臣行礼
  - 皇帝赐座

[Step 3] 赏赐
  - 皇帝赏赐群臣
  - 群臣谢恩

[Step 4] 宴会
  - 设宴款待群臣
  - 群臣敬酒

效果：
- 所有臣子忠诚度+3
- 民心+5
```

## 决策选项

每个节日提供2-4个选项：
- 举办庆典
- 赐宴群臣
- 赏赐妃子
- 大赦天下
- 无视节日

## 决策后果

| 决策 | 忠诚度/好感度 | 民心 | 经济 |
|------|--------------|------|------|
| 举办庆典 | +3（所有） | +5 | -10 |
| 赐宴群臣 | +2（臣子） | +3 | -5 |
| 赏赐妃子 | +3（妃子） | 0 | -3 |
| 大赦天下 | 0 | +10 | 0 |
| 无视节日 | -2（所有） | -5 | 0 |
