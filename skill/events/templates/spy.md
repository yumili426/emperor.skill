# 密探事件模板

## 场景

密探汇报情报。

## 事件类型

### 1. 忠诚度变化

```
场景：密探发现某人忠诚度变化

流程：
[Step 1] 密探来报
  "锦衣卫来报，首辅近日与户部尚书密会三次。"

[Step 2] 皇帝决策
  - 信任首辅
  - 怀疑首辅
  - 深入调查
  - 无视

[Step 3] 结果
  - 忠诚度变化
  - 可能触发后续事件
```

### 2. 秘密发现

```
场景：密探发现某人的秘密

流程：
[Step 1] 密探来报
  "东厂查实，将军私通北元，有书信为证。"

[Step 2] 皇帝决策
  - 逮捕将军
  - 警告将军
  - 信任将军
  - 无视

[Step 3] 结果
  - 根据决策更新关系
  - 可能触发叛变
```

### 3. 背叛发现

```
场景：密探发现背叛行为

流程：
[Step 1] 密探来报
  "西厂探知，贵妃暗中联络太后，似有异心。"

[Step 2] 皇帝决策
  - 逮捕贵妃
  - 警告贵妃
  - 信任贵妃
  - 无视

[Step 3] 结果
  - 根据决策更新关系
  - 可能触发后宫事件
```

### 4. 阴谋发现

```
场景：密探发现阴谋

流程：
[Step 1] 密探来报
  "锦衣卫来报，二皇子暗中结交大臣，似有夺嫡之心。"

[Step 2] 皇帝决策
  - 警告二皇子
  - 逮捕二皇子
  - 支持二皇子
  - 无视

[Step 3] 结果
  - 根据决策更新关系
  - 可能触发东宫事件
```

### 5. 假情报

```
场景：密探提供假情报

流程：
[Step 1] 密探来报
  "东厂来报，首辅贪污受贿。"

[Step 2] 皇帝决策
  - 相信密探
  - 怀疑密探
  - 调查首辅
  - 无视

[Step 3] 结果
  - 如果首辅清白：密探忠诚度-10
  - 如果首辅贪污：首辅忠诚度-20
```

## 密探忠诚度

密探也有忠诚度，可能骗你：

```
密探忠诚度 > 80：提供真实情报
密探忠诚度 50-80：可能提供假情报
密探忠诚度 < 50：经常提供假情报
```

## 决策后果

| 决策 | 忠诚度变化 | 其他影响 |
|------|-----------|---------|
| 信任 | +10 | 关系改善 |
| 怀疑 | -5 | 关系紧张 |
| 逮捕 | -30 | 该角色进入诏狱 |
| 警告 | -10 | 关系变化 |
| 调查 | 0 | 可能发现真相 |
| 无视 | 0 | 无变化 |
