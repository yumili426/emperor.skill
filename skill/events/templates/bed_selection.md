# 翻牌子模板

## 场景

皇帝选择今晚陪伴的妃子，体验后宫生活。

## 流程

```
[Step 1] 太监呈牌
  "启禀万岁，今晚翻谁的牌子？"

[Step 2] 展示牌子
  列出所有后宫妃子的牌子（姓名 + 等级 + 好感度 + 特殊状态）

[Step 3] 皇帝翻牌
  选择一个妃子（输入序号或姓名）

[Step 4] 互动对话
  与被选妃子进入深度互动
  - 根据妃子人设生成对话
  - 根据当前国家状况调整话题
  - 蒸馏角色使用灵魂原声

[Step 5] 结算
  - 被选妃子好感度+5
  - 其他妃子好感度-2
  - 可能触发特殊事件
```

## 好感度等级

| 好感度范围 | 等级 | 互动效果 |
|-----------|------|---------|
| 90-100 | 深情 | 触发深情对话，可能怀孕 |
| 70-89 | 喜爱 | 正常互动，关系加深 |
| 50-69 | 平淡 | 普通互动 |
| 30-49 | 冷淡 | 对话敷衍，可能冷场 |
| 0-29 | 厌恶 | 拒绝互动，可能引发冲突 |

## 互动对话类型

### 1. 深情互动（好感度 90+）

```
场景：妃子对皇帝深情款款

流程：
[Step 1] 妃子迎接
  "陛下~臣妾等你好久了。"

[Step 2] 深度对话
  - 妃子分享心事
  - 皇帝回应
  - 情感交流

[Step 3] 结果
  - 好感度+5
  - 可能触发怀孕事件
  - 稳定度+2
```

### 2. 日常互动（好感度 50-89）

```
场景：妃子与皇帝日常相处

流程：
[Step 1] 妃子迎接
  "陛下驾到，臣妾有失远迎。"

[Step 2] 日常对话
  - 妃子汇报后宫事务
  - 皇帝询问
  - 轻松交流

[Step 3] 结果
  - 好感度+5
  - 可能触发后宫事件
```

### 3. 冷淡互动（好感度 30-49）

```
场景：妃子对皇帝态度冷淡

流程：
[Step 1] 妃子迎接
  "陛下。"（简短行礼）

[Step 2] 尴尬对话
  - 妃子敷衍
  - 皇帝试图缓和
  - 气氛尴尬

[Step 3] 结果
  - 好感度+2（勉强）
  - 可能触发妃子不满事件
```

### 4. 冲突互动（好感度 0-29）

```
场景：妃子对皇帝有怨气

流程：
[Step 1] 妃子冷脸
  "陛下今夜怎么有空来臣妾这里？"

[Step 2] 冲突对话
  - 妃子表达不满
  - 皇帝选择应对：
    A. 安慰（好感度+5）
    B. 训斥（好感度-10，稳定度-2）
    C. 离开（好感度-5，其他妃子好感度+2）

[Step 3] 结果
  - 根据选择更新状态
```

## 特殊状态处理

### 有孕妃子

```
太监："启禀万岁，{妃子名}已有身孕，不宜侍寝。"

选项：
A. 换人（重新选择）
B. 照顾（前往探望，好感度+8，不翻牌子）
```

### 生病妃子

```
太监："启禀万岁，{妃子名}偶感风寒，正在静养。"

选项：
A. 换人（重新选择）
B. 探病（前往探望，好感度+10，不翻牌子）
```

### 禁足妃子

```
太监："启禀万岁，{妃子名}已被禁足，不宜侍寝。"

选项：
A. 换人（重新选择）
B. 解禁（解除禁足，好感度+15，稳定度-2）
```

## 触发特殊事件

翻牌子后可能触发以下事件：

| 事件 | 触发条件 | 效果 |
|------|---------|------|
| 妃子争宠 | 有两个妃子好感度差距 > 30 | 触发争宠事件 |
| 怀孕 | 好感度 90+ 且翻牌子 3 次以上 | 有概率怀孕 |
| 后宫阴谋 | 妃子关系紧张 | 可能触发阴谋事件 |
| 深夜密谈 | 好感度 80+ | 妃子分享秘密 |
| 情报传递 | 妃子有密探背景 | 传递情报 |

## 翻牌子限制

| 限制 | 说明 |
|------|------|
| 每日一次 | 每天只能翻一次牌子 |
| 不能重复 | 不能连续两天翻同一人 |
| 有孕/生病/禁足 | 不能翻这些妃子的牌子 |
| 后宫无人 | 如果后宫空虚，无法翻牌子 |

## 与后宫事件的关系

翻牌子是后宫系统的核心入口：

```
翻牌子
  ↓
互动对话
  ↓
好感度变化
  ↓
可能触发：
  - 妃子争宠
  - 怀孕生子
  - 后宫阴谋
  - 节日互动
```
