# 选秀模板

## 场景

礼部尚书主持选秀，皇帝选妃充盈后宫。

## 触发方式

- 皇帝输入 `/选秀`
- 皇后推荐（随机触发）
- 后宫空虚时自动建议

## 流程

```
[Step 1] 礼部上疏
  "礼部尚书上疏，奏请选秀充盈后宫。"

[Step 2] 展示候选
  系统生成 3-5 名候选女子

[Step 3] 皇帝决策
  选择纳入、遣返或推迟

[Step 4] 纳入后宫
  对选中的女子建立角色档案
```

## 候选生成规则

### 属性随机

| 属性 | 选项池 | 选择规则 |
|------|--------|---------|
| 姓氏 | 赵、钱、孙、李、周、吴、郑、王、冯、陈、褚、卫、蒋、沈、韩、杨 | 随机选 1 |
| 名字 | 婉儿、静娴、淑慧、若兰、紫嫣、雪儿、灵儿、梦瑶、诗韵、清雅 | 随机选 1 |
| 出身 | 官员之女、民间良家、藩属进贡、商贾之女、书香门第 | 随机选 1 |
| 才艺 | 琴、棋、书、画、歌、舞、女红、厨艺 | 随机选 1-2 |
| 相貌 | 普通(30%)、清秀(40%)、美丽(25%)、绝色(5%) | 按权重随机 |
| 性格 | 温柔、活泼、端庄、聪慧、冷傲、娇憨 | 随机选 1 |

### 特殊候选

每次选秀有 10% 概率出现"特殊候选"：

| 特殊属性 | 效果 | 触发事件 |
|---------|------|---------|
| 暗藏心机 | 忠诚度下降速度加倍 | 可能触发密探事件 |
| 天生丽质 | 好感度增长速度加倍 | 更容易获得宠爱 |
| 贤良淑德 | 后宫管理加成 | 可协助皇后管理后宫 |
| 祸国殃民 | 触发特殊事件概率增加 | 可能引发后宫动荡 |

## 皇帝决策选项

### A. 全部纳入

```
效果：
- 所有候选纳入后宫
- 初始等级：贵人
- 初始好感度：30-50（随机）
- 民心 +3
- 礼部尚书忠诚度 +5
```

### B. 选择其中几位

```
效果：
- 被选中的候选纳入后宫
- 被选者好感度 +10
- 未被选者好感度 -5
- 民心 +1
```

### C. 全部遣返

```
效果：
- 所有候选遣返回家
- 民心 -5
- 礼部尚书忠诚度 -10
- 可能触发礼部不满事件
```

### D. 选秀推迟

```
效果：
- 本次选秀取消
- 无变化
- 下次选秀概率 +10%
```

## 纳入后宫流程

对于被选中的女子：

```
[Step 1] 建立档案
  在 court/{slug}/ 目录下创建角色文件：
  - portrait.md: 角色画像
  - record.md: 朝堂档案
  - meta.json: 元数据（包含以下字段）
    - slug: 随机生成（如 "consort-xxx"）
    - name: 候选姓名
    - title: 贵人
    - court_role: 嫔妃
    - harem_level: 贵人
    - distillation_level: L1（系统生成）
    - loyalty: 50
    - affection: 初始好感度

[Step 2] 更新 kingdom.json
  将新角色添加到 harem 部分

[Step 3] 更新 court/index.md
  添加新角色到索引

[Step 4] 通知皇帝
  "选秀完成，{姓名}已入后宫，封为贵人。"
```

## 选秀限制

| 限制 | 说明 |
|------|------|
| 每年一次 | 每年只能选秀一次 |
| 后宫上限 | 后宫人数不超过 12 人 |
| 国力要求 | 民心 > 50 才能选秀 |
| 特殊时期 | 战争期间不能选秀 |

## 与现有系统的关系

- **翻牌子系统**：选秀后新妃子可被翻牌子
- **后宫事件系统**：新妃子可能触发争宠、阴谋等事件
- **密探系统**：特殊候选可能触发密探事件
- **角色成长系统**：新妃子从 1 级开始成长
