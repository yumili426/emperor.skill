# 社交场景映射

## 映射规则

将社交相关的现实事件映射为宫廷事件。

## 映射表

### 朋友聚会

```
现实：朋友聚会
宫廷：宴请群臣
台词模板："今晚设宴，与众卿共饮。"
```

### 朋友诉苦

```
现实：朋友诉苦
宫廷：臣子求见
台词模板："御史求见，称有要事禀报。"
```

### 收到礼物

```
现实：收到礼物
宫廷：藩国进贡
台词模板："藩国遣使进贡，献上{礼物}。"
```

### 送礼物

```
现实：送礼物
宫廷：赏赐臣子
台词模板："陛下下旨，赏赐{某人}{礼物}。"
```

### 被邀请

```
现实：被邀请参加活动
宫廷：藩国来朝
台词模板："藩国国王遣使，请求朝见。"
```

### 邀请别人

```
现实：邀请别人参加活动
宫廷：召见臣子
台词模板："陛下下旨，召见{某人}。"
```

### 吵架

```
现实：与人吵架
宫廷：朝堂争执
台词模板："{大臣A}与{大臣B}在朝堂上争执。"
```

### 和好

```
现实：与人和好
宫廷：冰释前嫌
台词模板："{大臣A}与{大臣B}握手言和。"
```

### 背叛

```
现实：被朋友背叛
宫廷：谋反
台词模板："密探来报，{大臣}暗中谋反。"
```

### 被骗

```
现实：被朋友欺骗
宫廷：蒙蔽
台词模板："陛下，{大臣}所言皆是谎言。"
```

## 映射流程

```
[Step 1] 识别事件类型
  - 聚会/邀请 → 宴会事件
  - 诉苦/吵架 → 人际关系事件
  - 礼物/背叛 → 权力事件

[Step 2] 选择映射模板
  - 根据事件类型选择
  - 选择合适的角色填充

[Step 3] 生成宫廷事件
  - 用宫廷语言描述
  - 保留事件本质
  - 增加戏剧性

[Step 4] 提供决策选项
  - 给用户2-4个选择
  - 每个选择有不同后果
```

## 输出格式

```markdown
*{旁白描述}*

**{角色名}**：{宫廷化的台词}

---

**陛下需要决策**：

1. {选项A}
   - 后果：{描述}

2. {选项B}
   - 后果：{描述}
```
