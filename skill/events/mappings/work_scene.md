# 工作场景映射

## 映射规则

将工作相关的现实事件映射为宫廷事件。

## 映射表

### 加班

```
现实：同事加班
宫廷：边关吃紧，请求增兵
台词模板："首辅奏报：{敌国}犯边，请求增兵。"
```

### 被批评

```

宫廷：谏官弹劾
台词模板："左都御史上疏弹劾{某人}。"
```

### 升职

## 映射流程

```
[Step 1] 识别事件类型
  - 加班/被催促 → 危机事件
  - 被批评/被解雇 → 负面事件
  - 升职/好消息 → 正面事件
  - 团队合作/竞争 → 人际关系事件

[Step 2] 选择映射模板
  - 根据事件类型选择
  - 选择合适的角色填充

[Step 3] 生成宫廷事件
  - 用宫廷语言描述
  - 保留事件本质
  - 增加戏剧性

[Step 4] 提供决策选项
  - 给用户2-4个选择
  - 每个选择有不同后果
```

## 输出格式

```markdown
*{旁白描述}*

**{角色名}**：{宫廷化的台词}

---

**陛下需要决策**：

1. {选项A}
   - 后果：{描述}

2. {选项B}
   - 后果：{描述}
```
