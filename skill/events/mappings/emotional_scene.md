# 情感场景映射

## 映射规则

将情感相关的现实事件映射为宫廷事件。

## 映射表

### 暗恋对象发消息

```
现实：暗恋对象发消息
宫廷：妃子传书
台词模板："贵妃遣宫女送来一方手帕。"
```

### 恋爱

```
现实：恋爱

台词模板："今夜，陛下翻了{妃子}的牌子。"
```

## 映射流程

```
[Step 1] 识别事件类型
  - 暗恋/恋爱 → 后宫事件
  - 分手/吵架 → 冲突事件
  - 关心/冷落 → 情感事件

[Step 2] 选择映射模板
  - 根据事件类型选择
  - 选择合适的妃子填充

[Step 3] 生成宫廷事件
  - 用宫廷语言描述
  - 保留情感色彩
  - 增加戏剧性

[Step 4] 提供决策选项
  - 给用户2-4个选择
  - 每个选择有不同后果
```

## 输出格式

```markdown
*{旁白描述}*

**{妃子名}**：{宫廷化的台词}

---

**陛下需要决策**：

1. {选项A}
   - 后果：{描述}

2. {选项B}
   - 后果：{描述}
```

## 特殊处理

### 暗恋类
- 保持含蓄
- 用宫廷方式表达
- 给用户情感回应的空间

### 恋爱类
- 保持甜蜜
- 用宫廷方式表达
- 给用户互动的机会

### 冲突类
- 保持张力
- 用宫廷方式表达
- 给用户决策的机会
