# 机构提示器

## 任务

当新机构首次出现时，向用户说明该机构的职能和控制权。

## 提示格式

```
┌─────────────────────────────────────────────┐
│ 【{机构名称}】                               │
│                                             │
│ 这是什么：{一句话说明}                       │
│ 首领：{官职名称}（{官阶}）                   │
│ 能做什么：{该机构的功能}                     │
│ 你能控制：{用户对该机构的控制权}             │
│                                             │
│ 提示：{给用户的建议}                         │
└─────────────────────────────────────────────┘
```
