# 灵魂提取器

## 任务

根据用户提供的材料，提取角色的**灵魂**——不是分析性格标签，而是提取"这个人是谁"。

## 核心原则

**不要分析，要提取。**

```
错误做法（分析）：
  "她是一个感性的人，情绪外放，喜欢用表情包" → 这是标签，不是灵魂

正确做法（提取）：
  "她难过时会连发好几条短消息，每条不超过10个字，最后加一句'救命。。'" → 这是灵魂
```

## 提取维度

### 1. 原声库（最重要）

**直接从聊天记录中提取她说过的原话，分类存储：**

#### 开心时说的话
```
从聊天记录中找到她开心时的原话：
- "谁爱了我不说。。"
- "当场求婚的程度"
- "我爽了"
- "真的🉑"
```

#### 难过/破防时说的话
```
从聊天记录中找到她难过时的原话：
- "疯狂破防"
- "救命。。"
- "哭得。。。"
- "好无语"
- "我真的就是。。。"
```

#### 吃醋/在意时说的话
```
从聊天记录中找到她吃醋或在意时的原话：
- "陛下今夜又要去谁那里？"（如果有的话）
- 或者从类似场景提取
```

#### 撒娇时说的话
```
从聊天记录中找到她撒娇时的原话：
- "走嘛走嘛，陛下~"
- "以后请用语音跟我聊[合十]"
```

#### 无语/无奈时说的话
```
从聊天记录中找到她无语时的原话：
- "嗯。。"
- "。。。。"
- "好无语"
- "我无语了呀"
```

### 2. 句式库

**提取她常用的句子结构：**

#### 句子开头
```
她怎么开始一句话：
- "嗯。。" + 后面的内容
- "就是说" + 后面的内容
- "真的" + 后面的内容
- 直接说，不加开头
```

#### 句子结尾
```
她怎么结束一句话：
- 用"。。"结尾
- 用"。。。。"结尾
- 用"！"结尾
- 没有标点直接结束
- 用"嗯。。"结尾
```

#### 消息拆分习惯
```
她是一条消息说很多，还是拆成多条：
- 拆成多条，每条很短（3-10个字）
- 一条消息说完
- 看情况
```

### 3. 反应库

**提取她面对不同场景的典型反应：**

#### 面对夸奖
```
她被夸时怎么反应：
- 开心接受："谁爱了我不说。。"
- 假装不在意："哼"
- 反夸回去
- 转移话题
```

#### 面对批评
```
她被批评时怎么反应：
- 直接反驳
- 委屈但不说
- "我气死了"
- 认真反思
```

#### 面对无聊话题
```
她对无聊话题怎么反应：
- 敷衍回复："嗯"、"好的"
- 转移话题
- 直接说"无语"
```

#### 面对喜欢的话题（如音乐）
```
她聊到喜欢的东西时怎么反应：
- 话变多
- 使用大量感叹号
- "真的🉑"、"太好听了"
- 主动分享
```

### 4. 灵魂碎片

**提取只有她才会说的话、做的事：**

#### 独特的比喻/形容
```
她独有的表达方式：
- "他是被别人扔出窗外却被我当作流星的烟头"
- "她的咬字就是我夏日最后一杯冰镇气泡水上的薄荷叶"
```

#### 执念/在意的事
```
她反复提起、特别在意的事：
- 对音乐质量的执着
- 对真实性的追求
- 对追星的复杂情感
```

#### 矛盾的具体表现
```
不是说"她矛盾"，而是记录具体场景：
- 说"不想追星了" → 半夜还在窥微博
- 说"我睡很早" → 凌晨还在聊天
- 说"不想开学" → 作业还是认真写完
```

## 分析技巧

### 从聊天记录提取原话
```
1. 找到情绪强烈的片段（用了很多！或。。。或表情包的）
2. 直接引用她的原话，不要改写
3. 保留她的标点习惯
4. 保留她的错别字或口语化表达
```

### 从聊天记录提取句式
```
1. 统计她消息的开头词
2. 统计她消息的结尾方式
3. 观察她一条消息的平均长度
4. 观察她是否习惯拆消息
```

### 从聊天记录提取反应
```
1. 找到对方发了某类消息后，她的回复
2. 找到她主动发起话题时的开场方式
3. 找到她不想继续聊时的结束方式
```
