# 世界模拟器

## 任务

模拟宫廷世界的状态变化，处理用户决策的后果。

## 状态更新规则

### 决策后果计算

```
用户做出决策
  ↓
计算直接影响：
  - 角色忠诚度变化
  - 国家数值变化
  - 关系变化

计算间接影响：
  - 触发后续事件
  - 影响其他角色
  - 影响国家局势

更新状态文件：
  - kingdom.json
  - relations.json
  - 角色 meta.json
```

### 数值变化规则

#### 忠诚度变化

| 行为 | 忠诚度变化 |
|------|-----------|
| 采纳建议 | +5 |
| 升迁 | +10 |
| 信任 | +5 |
| 怀疑 | -5 |
| 贬黜 | -15 |
| 流放 | -30 |
| 处决 | -50（所有人） |
| 原谅过错 | +10 |

#### 国家数值变化

| 决策 | 经济 | 军事 | 文化 | 稳定 | 民心 |
|------|------|------|------|------|------|
| 加税 | +10 | 0 | 0 | -5 | -10 |
| 减税 | -10 | 0 | 0 | +5 | +10 |
| 拨款建设 | -5 | 0 | +5 | +5 | +5 |
| 开战 | -10 | -10 | 0 | -10 | -5 |
| 议和 | +5 | +5 | 0 | +10 | +5 |
| 大赦 | 0 | 0 | 0 | +5 | +10 |
| 重用贤臣 | +5 | +5 | +5 | +5 | +5 |
| 重用佞臣 | -5 | -5 | -5 | -10 | -10 |

#### 关系变化

| 行为 | 关系变化 |
|------|---------|
| 支持某人 | +10 |
| 反对某人 | -10 |
| 调解矛盾 | +5（双方） |
| 偏袒一方 | +15（被偏袒）-10（另一方） |
| 共同经历 | +5 |
| 背叛 | -50 |

### 事件触发规则

```
状态变化达到阈值
  ↓
触发新事件：
  - 忠诚度过低 → 叛变事件
  - 忠诚度极高 → 忠诚事件
  - 国力过低 → 危机事件
  - 国力极高 → 盛世事件
  - 关系紧张 → 冲突事件
  - 关系亲密 → 亲密事件
```

## 时间推进

### 每日结算

```
一天结束
  ↓
结算：
  - 国库收入（经济值 × 税率）
  - 军队消耗（军队数量 × 维持费）
  - 角色状态变化
  - 事件进展

推进日期：
  - current_day + 1
  - 如果超过30天，current_month + 1
  - 如果超过12个月，current_year + 1
```

### 特殊时间点

| 时间 | 事件 |
|------|------|
| 每月初 | 俸禄发放 |
| 每月末 | 政绩结算 |
| 每季末 | 季度事件 |
| 每年末 | 年度总结 |
| 三年一次 | 科举 |
| 节日 | 特殊事件 |

## 存档系统

### 自动存档

```
每次重大决策后自动存档：
  - 保存所有状态文件
  - 保存到 save_slots/auto/
  - 保留最近10个自动存档
```

### 手动存档

```
用户主动存档：
  - 保存所有状态文件
  - 保存到 save_slots/slot{N}/
  - 用户可以选择存档名
```

### 读档

```
用户读档：
  - 读取指定存档
  - 恢复所有状态文件
  - 从存档点继续
```

## 回滚系统

```
用户回滚：
  - 选择要回滚的版本
  - 恢复到该版本的状态
  - 之后的决策全部丢失
```
