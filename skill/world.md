# 世界设定

## 架空大明

这是一个融合明代与唐代制度的架空世界。政治、军事、外交均以明朝为蓝本，但剧情完全架空。

---

## 国家概况

- 国号：大明
- 年号：由陛下登基时确立
- 都城：京师（北京）
- 领土：中原、江南、蜀中、北境、西疆、南疆

---

## 核心机构

### 六部（你的政策面板）

| 部门 | 职能 | 你能做什么 |
|------|------|-----------|
| 吏部 | 管官员 | 升迁/贬黜/考核 |
| 户部 | 管钱 | 加税/减税/拨款 |
| 礼部 | 管礼仪外交 | 派使臣/办选秀 |
| 兵部 | 管军事 | 调兵/开战/议和 |
| 刑部 | 管司法 | 审案/赦免/处决 |
| 工部 | 管建设 | 修城/治水/造兵器 |

### 内阁（你的智囊团）

- 首辅：首席顾问，权力最大
- 次辅：二号人物
- 群辅：3-5人参与政务

### 后宫（你的后花园）

等级：皇后 → 贵妃 → 妃 → 嫔 → 贵人

后宫管理由皇后负责，日常事务自动化处理，皇帝只需关注关键决策。

### 东宫（你的继承人）

- 太子：你选的继承人
- 詹事府：太子的属官
- 皇子：可能争夺储位

### 密探（你的眼睛和耳朵）

- 锦衣卫：皇帝亲军，最忠诚
- 东厂：宦官掌控，监控官员
- 西厂：权力最大，连东厂都能查

## 邻国

### 北元
- 关系：敌对
- 威胁：主要军事威胁
- 特点：骑兵强大，经常犯边

### 朝鲜
- 关系：朝贡国
- 态度：最忠诚的藩属
- 特点："事大"外交，频繁朝贡

## 科举系统

三年一届：

```
乡试（秋闱）→ 会试（春闱）→ 殿试
    ↓              ↓           ↓
  举人           贡士        进士
 解元           会元        状元
```

进士排名：
- 一甲（3人）：状元、榜眼、探花 — 赐"进士及第"
- 二甲：赐"进士出身"
- 三甲：赐"同进士出身"
