## 安装

### Claude Code

> **重要**：Claude Code 从 **git 仓库根目录** 的 `.claude/skills/` 查找 skill。请在正确的位置执行。

```bash
# 安装到当前项目（在 git 仓库根目录执行）
mkdir -p .claude/skills
git clone https://github.com/yumili426/emperor.skill .claude/skills/emperor

# 或安装到全局（所有项目都能用）
git clone https://github.com/yumili426/emperor.skill ~/.claude/skills/emperor
```

### Kiro

```bash
git clone https://github.com/yumili426/emperor.skill ~/.kiro/skills/emperor
```

### 其他平台

`skill/` 目录下的内容完全平台无关。只需让目标平台加载 `skill/SKILL.md` 作为入口即可。

### 依赖（可选）

```bash
pip install -r requirements.txt
```

### 环境要求

- 上述任意一个 AI 编程助手
- Python 3.9+（可选，用于聊天记录解析）
- 不需要 GPU、本地模型或 Docker

## 使用

在 Claude Code 中输入：

```
/emperor
```

首次启动会引导你完成：选择皇帝人设 → 确立年号 → 蒸馏第一个角色 → 进入游戏。

### 游戏命令

| 命令 | 说明 |
|------|------|
| `/早朝` | 查看今日事件，做决策 |
| `/翻牌子` | 选择妃子互动 |
| `/奏折` | 处理政务 |
| `/蒸馏` | 蒸馏新人物入宫 |
| `/选秀` | 选秀充盈后宫 |
| `/召见 {角色名}` | 召见某位角色对话 |
| `/密探` | 派遣密探监视某人 |
| `/宫廷` | 查看所有角色关系图谱 |
| `/东宫` | 管理太子和皇子 |
| `/地图` | 查看疆域和邻国 |
| `/存档` | 保存/读取游戏进度 |
| `/版本` | 版本管理 |
| `/退出` | 退出游戏 |

## 功能特性

### 皇帝人设

| 人设 | 名称 | 互动风格 |
|------|------|---------|
| 勤政明君 | 圣天子 | 决策谨慎，重用贤臣 |
| 咸鱼躺帝 | 逍遥帝 | 能不管就不管，被动触发事件 |
| 腹黑帝王 | 暗龙帝 | 多疑，善于制衡 |
| 战争狂人 | 征天帝 | 好战，武力优先 |
| 风流天子 | 情深帝 | 重情感，偏后宫线 |

### 蒸馏系统

把身边真实的人蒸馏成宫廷角色。支持三级蒸馏：

| 级别 | 名称 | 所需材料 | 角色质量 |
|------|------|---------|---------|
| L1 | 微笔速写 | 用户口述（5分钟） | 模板化，够用 |
| L2 | 工笔小像 | 聊天记录 + 口述（15分钟） | 有灵魂原声 |
| L3 | 传神写照 | 完整聊天记录 + 照片 + 口述（30分钟+） | 深度灵魂 |

核心理念：**灵魂提取，不是性格分析**。提取"她难过时连发好几条短消息，最后加一句'救命。。'"，而不是"感性、活泼、情绪外放"。

### 核心系统

| 系统 | 说明 |
|------|------|
| 早朝系统 | 每日事件播报，皇帝决策 |
| 后宫系统 | 翻牌子、妃子争宠、后宫阴谋 |
| 密探系统 | 锦衣卫 / 东厂 / 西厂，监视任何人 |
| 东宫系统 | 太子培养、皇子争储 |
| 开疆扩土 | 外交博弈、战争系统 |
| 隐喻映射 | 现实事件自动转化为宫廷事件 |

### 隐喻映射

现实事件会自动转化为宫廷事件：

| 现实场景 | 宫廷映射 |
|---------|---------|
| 同事加班 | 边关吃紧 |
| 朋友诉苦 | 臣子求见 |
| 暗恋对象发消息 | 妃子传书 |
| 老师布置作业 | 翰林进谏 |
| 被老板批评 | 谏官弹劾 |

## 注意事项

* **蒸馏材料质量决定还原度**：聊天记录 + 口述 > 仅口述
* 建议优先提供：
  1. **深夜对话** — 最能暴露真实性格
  2. **情绪波动的记录** — 生气、难过、兴奋时的表达
  3. **争吵记录** — 冲突时的原话最有灵魂
* 所有蒸馏数据仅存储在本地，不会上传任何服务器
* 不会主动联系任何真人，生成的虚拟角色均有明确标识
* 可随时删除任何角色数据
